## Typographic scale

Our typography uses a single scale for use with all fonts. The scale consists of font-size and line-height combinations, as shown below.

| Scale | Font Size     | Line Height |
| ----- |:-------------:| -----------:|
| -2    | 12px          | 16px |
| -1    | 14px          | 16px |
| 0     | 16px          | 20px |
| 1     | 18px          | 20px |
| 2     | 20px          | 24px |
| 3     | 24px          | 28px |
| 4     | 28px          | 32px |
| 5     | 32px          | 32px |
| 6     | 40px          | 40px |
| 7     | 48px          | 48px |
| 8     | 56px          | 56px |
| 9     | 72px          | 72px |
| 10    | 84px          | 84px |
